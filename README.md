# Job Tracker

A full-stack web application that helps job seekers manage applications, interviews, and offers in one place.

Built with React, Spring Boot, PostgreSQL, and Docker, the application provides a simple way to track opportunities, monitor progress, and stay organised throughout the job search process.

## Features

- Create, edit, and delete job applications
- Track application status (Saved, Applied, Interview, Offer, Rejected)
- Search applications by company
- Filter applications by status
- Dashboard statistics and progress tracking
- RESTful API built with Spring Boot
- PostgreSQL database integration
- Dockerised development environment

## Why I Built This

While applying for graduate roles, I found it difficult to keep track of applications, interviews, deadlines, and offers. I initially used spreadsheets, but found them time-consuming to maintain and not ideal for managing the entire application process.

To solve this problem, I decided to build my own solution and create a central place to manage every stage of the job search journey.

## Product Development Process

After building the initial prototype, I invited three students and recent graduates to test the application and provide feedback.

Their feedback highlighted several improvement opportunities, including:

- Better visibility of application statuses
- More compact application cards
- Resume autofill to reduce manual data entry
- Reminders for saved applications
- Improved notifications and user feedback

The project is currently being refined based on these insights as part of an ongoing iterative development process.
